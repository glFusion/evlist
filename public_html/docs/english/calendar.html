glFusion Evlist Plugin Configuration
glFusion Wiki
Calendar Name
Enter a name for this calendar. Names should be unique.
Colors
Select the foreground and background colors used to display events for this calendar within calendar views.
Icon
Enter an option icon name to use with this calendar. Enter only the unique name, do not include "uk-icon-" or "fa-"
Enabled
Check to enable this calendar. Disabling a calendar will prevent its events from being shown on any calendar view as well as removing it from the selection during event entry.
ICal Enabled
When this box is checked, this calendar will be included in iCal feeds. If this option is disabled after a feed has been published, events will be excluded from future feeds, but not necessarily removed from feed consumers that have already included them.
Show in Upcoming Events?
Check this box to have events from this calendar shown in upcoming event blocks. Unchecking will prevent them from displaying regardless of the setting set for the event.
Show After...
Select the position for this calendar in the selection list
Owner
This is for information only to remind you that all calendars are owned by "Admin"
Group
Select the group associated with this calendar. This group is used in the permissions matrix below.
Permissions
For calendars, permissions have the following meanings: Read/Write - users may view events in this calendar and submit events to it. Read Only - users may view events in this calendar. No access - the calendar is hidden from the users or group.
