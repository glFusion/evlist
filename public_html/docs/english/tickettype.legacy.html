EvList Plugin

EvList Plugin - Ticket Types Help

General Information
Option Description
Description Enter a short description for this ticket, such as "General Admission", "Balcony"e;, etc. This will appear on the printed ticket.
Event Pass Check this box if this ticket type is an event pass that covers all recurring instances. If left unchecked, the purchase of this ticket type will cover only one instance.
Enabled The ticket type can be enabled or disabled by checking or unchecking this box. Ticket types cannot be deleted if they are associated with any purchased tickets, but can be hidden from view by disabling them.
